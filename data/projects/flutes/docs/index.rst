Installation
============

Install stable release from PyPI:

.. code:: bash

    pip install flutes

Or, install the latest commit from GitHub:

.. code:: bash

    pip install git+https://github.com/huzecong/flutes.git

Usage
=====

Flutes provides a variety of utility functions. Take a look at the
`examples <https://github.com/huzecong/flutes/blob/master/examples>`_ folder to see what you can do with it!

Please refer to the `API documentations <api.html>`_ for more details.
